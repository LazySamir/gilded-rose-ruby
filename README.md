# Gilded Rose Ruby
Refactored legacy ruby code by writing tests first, then extracting logic from a single method, into discreet methods.

### How to install

```
$ git clone git@github.com:LazySamir/gilded-rose-ruby.git
$ cd gilded-rose-ruby
$ bundle 
```

### How to test

```
$ rspec
```

### How to run

```
$ ruby texttest_fixture.rb
```
